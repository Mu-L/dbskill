dontbesilent / dbskill

把眼前的问题，推进成下一步行动。

dbskill 是面向创业者与内容创作者的中文 AI Skills 工具箱，覆盖商业诊断、内容创作、行动推进与长期决策。

前往 GitHub 查看使用手册

开始使用

在支持 Skills 的 Agent 中安装全部工具：

npx -y skills add dontbesilent2025/dbskill -g --all

你可以从这些问题开始

  • 产品、定价与客户判断
  • 选题、标题与短视频文稿
  • 执行卡点与行动设计
  • 决策记录与长期复盘
